Chapter 4: Church Numerals: Decrement

Predecessor​

The implementation of the predecessor function is much harder than previous ones. In fact, Church thought for a long time that it might not be possible, until his student Kleene found it. In fact, there is a legend that Kleene conceived the idea while visiting his dentist, which is why the trick is called the wisdom tooth trick.

Wisdom Tooth Trick​

Let's define a helper function Φ\PhiΦ:

Φ=λp.π(S(π1p))(π1p)\Phi=\lambda p.\pi(S(\pi_1p))(\pi_1p)Φ=λp.π(S(π1​p))(π1​p)

Φ(π0 0)=(π1 0)\Phi(\pi0\space0)=(\pi1\space0)Φ(π0 0)=(π1 0)

By means of mathematical induction, we get:

Φn(π0 0)=(πn (n−1))\Phi^n(\pi0\space0)=(\pi n\space (n-1))Φn(π0 0)=(πn (n−1))

With the help of ordered pair, predecessor function could be defined as:

pred=λn.π2(nΦ(π0 0))pred n=n−1\text{pred}=\lambda n.\pi_2(n\Phi(\pi0\space 0))\\ \text{pred }n=n-1pred=λn.π2​(nΦ(π0 0))pred n=n−1

Note that pred 0=0\text{pred }0=0pred 0=0. It is a limit of church numerals. Is it possible to define negative numbers or even rational numbers? Of course! Remember what I mentioned in Data Abstraction part? Build one if it doesn't exist yet!

Subtraction​

Subtraction compared to predecessor is similar to addition compared to successuor.

sub=λmn.n pred msub mn=m−n\text{sub}=\lambda mn.n\text{ pred }m\\ \text{sub }mn=m-nsub=λmn.n pred msub mn=m−n

It reads as if "evaluating nthn^\text{th}nth predecessor of mmm".

Proof is similar to the addition operation.

Note: if m<nm<nm<n, sub mn=0\text{sub }mn=0sub mn=0 according to the definition of predecessor function.
