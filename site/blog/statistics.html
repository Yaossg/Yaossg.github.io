四个统计学分布​

正态分布​

X∼N(μ,σ2)E(X)=μ,D(X)=σ2X\sim N(\mu, \sigma^2)\\ E(X)=\mu,D(X)=\sigma^2X∼N(μ,σ2)E(X)=μ,D(X)=σ2

卡方分布​

χ2=∑i=1nXi2∼χ2(n) 其中 X∼N(0,1)E(χ2)=n,D(χ2)=2n\chi^2=\sum_{i=1}^n X_i^2\sim\chi^2(n)\\ \text{ 其中 } X \sim N(0, 1)\\ E(\chi^2)=n,D(\chi^2)=2nχ2=i=1∑n​Xi2​∼χ2(n) 其中 X∼N(0,1)E(χ2)=n,D(χ2)=2n

t 分布​

又叫学生分布

T=XY/n∼t(n) 其中 X∼N(0,1),Y∼χ2(n)T=\dfrac{X}{\sqrt{Y/n}} \sim t(n)\\ \text{ 其中 } X \sim N(0, 1), Y\sim\chi^2(n)T=Y/n​X​∼t(n) 其中 X∼N(0,1),Y∼χ2(n)

F 分布​

又叫费希尔（Fisher）分布

F=X/n1Y/n2∼F(n1,n2) 其中 X∼χ2(n1),Y∼χ2(n2)F=\dfrac{X/n_1}{Y/n_2}\sim F(n_1,n_2)\\ \text{ 其中 } X\sim\chi^2(n_1), Y\sim\chi^2(n_2)F=Y/n2​X/n1​​∼F(n1​,n2​) 其中 X∼χ2(n1​),Y∼χ2(n2​)

距估计​

μ^=X‾σ^2=n−1nS2\hat{\mu}=\overline{X}\\ \hat{\sigma}^2=\dfrac{n-1}nS^2μ^​=Xσ^2=nn−1​S2

极大似然估计​

核心思想是使似然函数 LLL 取极大值。即求出一个 θ\thetaθ，使取出这个样本的概率最大。

L=∏i=1nP(Xi=xi)ln⁡L=∑i=1nln⁡P(Xi=xi)∂ln⁡L∂θ=0L=\prod_{i=1}^nP(X_i=x_i)\\ \ln L=\sum_{i=1}^n \ln P(X_i=x_i)\\ \dfrac{\partial \ln L}{\partial\theta}=0L=i=1∏n​P(Xi​=xi​)lnL=i=1∑n​lnP(Xi​=xi​)∂θ∂lnL​=0

区间估计与假设检验​

  • 估计或检验 μ\muμ，当 σ\sigmaσ 已知时
U=X‾−μσ/n∼N(0,1)U=\dfrac{\overline{X}-\mu}{\sigma/\sqrt{n}}\sim N(0,1)U=σ/n​X−μ​∼N(0,1)
  • 估计或检验 μ\muμ，当 σ\sigmaσ 未知时
T=X‾−μS/n∼t(n−1)T=\dfrac{\overline{X}-\mu}{S/\sqrt{n}}\sim t(n-1)T=S/n​X−μ​∼t(n−1)
  • 估计或检验 σ\sigmaσ，当 μ\muμ 未知时
χ2=n−1σ2S2∼χ2(n−1)\chi^2=\dfrac{n-1}{\sigma^2}S^2\sim\chi^2(n-1)χ2=σ2n−1​S2∼χ2(n−1)
